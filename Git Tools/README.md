# Git Tools

Small helpers for Git and GitHub workflows.

## Included Scripts

- `list_merged_pr_commits.py`: Lists merged PRs between two tags or two commits for one or more repos, with PR links and titles.

## Usage

```bash
python3 "Git Tools/list_merged_pr_commits.py" <tag|commit> <start_ref> <end_ref> <repo_path_1> [repo_path_2 ...]
```
